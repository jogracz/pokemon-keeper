# pokemon-keeper
Fullstack application for keeping Pokemons 

MongoDB (Atlas), Express, React, Context API and Hooks.

I created this app following Brad Traversy's Udemy "React front to back' course and combining several projects.

App's abilities:
* register new user,
* login,
* logout,
* delete account,
* search for Pokemons,
* add a Pokemon,
* delete a Pokemon
